IDE e pacotes:
Neste trabalho nao foram usadas IDE's para o desenvolvimento, foi gerado um ambiente virtual

  - Matplotlib -> Utilizada para geração de graficos
  - Tkinter -> Utilizada para geração de interface gráfica
  - Nukita -> Utilizada para geração do executável
